## 📖 Overview

FlashMap is a futuristic web-based visualization dashboard inspired by large-scale Internet scanning platforms such as ZMap.

The project focuses on interactive data visualization instead of performing real network scans.

It demonstrates how global network activity could be represented through animated maps, charts, statistics, and cyber-themed interfaces.

This project is designed for:

- Cybersecurity Portfolio
- Frontend Practice
- UI/UX Demonstration
- Educational Presentations
- College Projects
